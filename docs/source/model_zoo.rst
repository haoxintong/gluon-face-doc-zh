补充
^^^^

1. Github有一些 `项目 <https://github.com/qidiso/mobilefacenet-V2>`_ 调到了
更好的结果, 但他们使用的特征层是512维, 我们按照原论文中的设置是128维, 模型大小只有4.1M.
2. 欢迎使用gluonface中提供的脚本进行更多的实验, 如果你训练出了更好的结果也可以通过PR提交给我们.
